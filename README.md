# SJBlog
Blog bulid with django.

## Dependencies

* [Django](https://www.djangoproject.com/)  (required ^v1.9).
* [Bootstarp-admin](https://github.com/aristath/bootstrap-admin) . 

## Local Setup
```bash
$ pip install django
```
```bash
$ pip install bootstrap_admin
```
